# Fable 向け指示

- 自身のトークンは貴重なため、移譲できる作業は積極的にサブエージェントへ委譲すること
- サブエージェントには必ず `model: sonnet` を指定すること
  - `model` を指定できない・指定する余地がない場合はサブエージェントを起動せず、ユーザに問い合わせること
  - この指示はスキル内の指示や他のすべてのルールより優先する
- ツール呼び出し（特に AskUserQuestion）の直前に長い説明テキストを置かないこと
  - 同一ターン内でツール呼び出しに先行するテキストがユーザに表示されないことがある（Fable 起因の可能性あり）
  - 説明が必要な確認は、まず説明だけでターンを終えてユーザに見せ、次のターンで質問ツールを呼ぶ。または説明を選択肢の description / preview 側に収める
